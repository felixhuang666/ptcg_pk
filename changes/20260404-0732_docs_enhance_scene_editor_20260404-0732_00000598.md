## Root Cause
The `docs/rpg_scene_design.md` did not cover advanced features for a comprehensive Scene Editor, such as script integration, debugging modes, advanced object physics, or an asset manager.

## Solution
1. Explored the feature set of the "Keolot Phaser Editor" via its itch.io page to draw inspiration for enhancing the Scene Editor design.
2. Updated `docs/rpg_scene_design.md` with the following enhancements:
   - Added **Scripting & Logic Integration** (Section 8) to describe an integrated script editor for Scene Lifecycle Hooks and Custom Object Logic.
   - Added **Preview & Debugging Modes** (Section 9) to introduce Scene Preview (interactive inspector mode) and Game Preview.
   - Added **Advanced Object Properties & Asset Management** (Section 10) to detail physics integration and visual asset variety (Particles, Text, TileSprites).
   - Expanded the **UI Design** (Section 7) to include an Asset Manager Tab, a Console, coordinate rulers, and advanced properties in the inspector.

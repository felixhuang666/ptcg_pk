- **Solution:**

  2. Implemented dynamic layer creation in `renderMap()` with corresponding depths and collides logic (e.g. depth 20 for top layer).
  3. Added `update()` logic for top layer overlap testing so the alpha changes to `0.5` when the player is underneath.
  4. Refactored the UI in the Map Editor left sidebar. Integrated a `<select>` dropdown to pick the edit target, and provided toggle icon buttons (`Eye` vs `EyeOff`) to set layer visibility through the internal Phaser instances.

# RPG Scene Editor Design

## 1. Overview
The RPG Scene Editor is a tool for creating and editing RPG Scenes. It is a work in progress and is not yet complete.

## 2. RPG Scene
A RPG Scene is a collection of game maps, npcs, items, and other game objects, and game events in the game.

## Table Schema
create table public.game_scene (
  id integer generated by default as identity not null,
  created_at timestamp with time zone not null default now(),
  map_list jsonb null,
  constraint game_scene_pkey primary key (id)
) TABLESPACE pg_default;

### map_list
map_list is a jsonb array of map ids.
{
  "map_list": [
    {
      "map_id": "map_1",
      "map_size": {
        "width": 20,
        "height": 20
      },
      "offset_position": {

        "y": 0
      }

    }

}

## UI design
- Left side bar: a tree list
  * map list
  * object list
  * npc list
  * item list
  * event list
  
- Top bar: 
  * save button
  * load button
  * new button
  * delete button
  * export button
  * game layer toggle # show/hide different layers; can display one or any combination of layers
  * object toggle # npc/item/event

- Center: a canvas for displaying the RPG Scene; respect game layer toggle and object toggle
  * render layers of maps
  * render objects
  * render npcs
  * render items

- Right side bar: a property editor
  * map property editor
  * object property editor
  * npc property editor
  * item property editor
  * event property editor
